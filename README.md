# Codeforces POTD Plus

A Chrome extension that enhances your Codeforces daily practice.

## Features
- **Global POTD**: A deterministic daily problem for all users.
- **Personalized POTD**: Problems tailored to your rating (+/- 200).
- **Daily Persistence**: Problems rotate every 24 hours at midnight.
- **Streak Tracking**: Track your daily solve streak.
- **Calendar**: Visual history of your solves.
- **Dark Mode UI**: Clean, modern interface.

## Installation
1. Clone this repo.
2. Run `npm install`.
3. Run `npm run build`.
4. Load `dist` folder in Chrome Extensions (Developer Mode).
